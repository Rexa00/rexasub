RexaSub

ابزاری برای استخراج خودکار لینک‌های VLESS / VMess و تولید فایل اشتراک آماده‌ی استفاده. این پروژه با الهام از سازوکارهای NOVA.PROXY ساخته شده است تا خروجی‌های قابل کپی و سازگار با همه پلتفرم‌ها تولید کند.

استخراج هوشمند

خواندن محتوا از URL اشتراک، تشخیص مثال‌های متنی و Base64، یافتن host، add، sni و تبدیل به لینک‌های قابل استفاده.

خروجی فارسی

ساخت فایل خروجی با توضیحات فارسی و زمان ثبت در تهران تا کاربرها بتوانند بدون دردسر آن را استفاده کنند.

انتشار خودکار

اجرای هر دقیقه با GitHub Actions و ثبت تغییرات در مخزن برای دسترسی همیشگی.

سازوکار جدید

این پروژه از مدل نوآوری NOVA.PROXY الهام گرفته و یک سازوکار مبتنی بر سه گام ایجاد کرده است:

  • خواندن و تحلیل محتوا از لینک اشتراک
  • استخراج عِناصری مثل host، sni، add و آی‌پی‌ها
  • ساختن خروجی کامل، قابل‌کپی و آماده برای استفاده در Hiddify، V2Ray، Clash و سایر اپ‌ها

به جای انتشار فقط آی‌پی، این پروژه یک خروجی اشتراک پویا تولید می‌کند که پل ارتباطی بین لینک‌های خام و مصرف نهایی کاربران است.

ویژگی‌های کلیدی

سازگار با همه پلتفرم‌ها

خروجی به فرمتی است که کاربران می‌توانند آن را کپی و در اپلیکیشن‌های مختلف قرار دهند.

زمان دقیق در کنار هر خروجی

هر لینک همراه با timestamp ثبت می‌شود تا کاربری و ردیابی آسان‌تر شود.

ساختار منظم و فارسی

تمام توضیحات، نظرات و راهنمایی‌ها به زبان فارسی نوشته شده است تا برای کاربران ایرانی مناسب باشد.

چگونه از پروژه استفاده کنیم

git clone https://github.com/Rexa00/rexasub.git
cd rexasub
python scripts/fetch_ips.py

برای اجرای خودکار در GitHub Actions، فایل workflow در مسیر .github/workflows/fetch-ips.yml موجود است.

بازخورد و توسعه

اگر دوست دارید، می‌توانیم سازوکار را گسترش دهیم تا خروجی‌های بیشتری مثل Clash YAML یا JSON ویژه تولید کند و صفحه مستندات را به یک داشبورد ساده تبدیل کنیم.

حالا یک صفحه توضیحی گرافیکی داریم که سازوکار «NOVA.PROXY الهام‌گرفته» را شرح می‌دهد. برای مشاهده فقط کافی‌ست فایل docs/index.html را باز کنید.
