# rexasub

این پروژه یک ابزار استخراج و انتشار اشتراک اشتراکی است که با الهام از سازوکارهای <strong>NOVA.PROXY</strong> ساخته شده است. RexaSub لینک‌های VLESS، VMess، Cloudflare و سایر اشتراک‌های پنهان را تحلیل می‌کند و خروجی آماده‌ی استفاده برای همه پلتفرم‌ها تولید می‌کند.

## صفحه توضیحی گرافیکی

یک نسخه‌ی گرافیکی و فارسی‌شده از توضیحات پروژه در مسیر زیر ساخته شده است:

- `docs/index.html`

این صفحه شامل:
- معرفی قابلیت‌ها
- سازوکار استخراج هوشمند
- توضیح نحوه استفاده
- نمایش سازوکار دقیق مشابه NOVA.PROXY

## امکانات

- دریافت خودکار محتوای لینک اشتراک از یک URL
- پشتیبانی از داده‌های متنی و Base64
- استخراج host، sni، add، server و آی‌پی‌ها از لینک‌های VLESS/VMess
- تولید خروجی آماده‌ی کپی و استفاده با زمان تکمیلی
- ذخیره‌سازی تاریخچه‌ی خروجی‌ها در `ips.txt`
- اجرای خودکار هر دقیقه با GitHub Actions

## ساختار پروژه

- `scripts/fetch_ips.py`: اسکریپت اصلی استخراج و تولید خروجی اشتراک
- `.github/workflows/fetch-ips.yml`: workflow برنامه‌ریزی‌شده برای اجرای خودکار هر دقیقه
- `tests/`: تست‌های منطق استخراج و پردازش
- `subscription.txt`: خروجی کپی‌پِیست آماده و فارسی‌شده
- `ips.txt`: تاریخچه‌ی کامل آی‌پی‌ها همراه با timestamp
- `docs/index.html`: صفحه توضیحی گرافیکی و فارسی برای معرفی پروژه

## نحوه اجرا

1. کلون کردن مخزن
   ```bash
   git clone https://github.com/Rexa00/rexasub.git
   cd rexasub
   ```

2. اجرای اسکریپت
   ```bash
   python scripts/fetch_ips.py
   ```

3. اجرای تست‌ها
   ```bash
   python -m unittest discover -s tests -v
   ```

## خروجی

- `subscription.txt`: خروجی آماده‌ی کپی با لینک‌های VLESS و زمان هر مورد
- `ips.txt`: تاریخچه‌ی کامل آی‌پی‌ها همراه با timestamp

هر سطر شامل زمان و لینک یا آی‌پی استخراج‌شده است.

## GitHub Actions

workflow در مسیر .github/workflows/fetch-ips.yml به‌صورت خودکار هر دقیقه اجرا می‌شود و در صورت وجود تغییر، آن را در مخزن ثبت می‌کند.

## نوآوری و سازوکار NOVA.PROXY-like

این پروژه از سازوکارهای NOVA.PROXY الهام گرفته است و به جای صرفاً ذخیره آی‌پی، یک خروجی اشتراک قابل استفاده تولید می‌کند. مرحله‌های این سازوکار به شرح زیر است:

1. دریافت محتوا از لینک اشتراک
2. استخراج اطلاعات host، sni، add و آدرس‌های IP
3. تولید خروجی آماده‌ی کپی و استفاده در همه پلتفرم‌ها

## نکته

اگر لینک اشتراک شما شامل دامنه‌های Cloudflare یا hostnames باشد، اسکریپت تلاش می‌کند آن‌ها را به آی‌پی حل کند و در خروجی ثبت نماید.
